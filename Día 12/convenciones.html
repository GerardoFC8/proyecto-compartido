Explora las Convenciones

Las convenciones de nomenclatura son esenciales para escribir código limpio, legible y mantenible. Haz clic en las pestañas a continuación para aprender sobre cada una, ver su formato, sus casos de uso principales y ejemplos prácticos.

Camel Case ejemploDeVariable

Formato

La primera letra de la primera palabra es minúscula; la primera letra de cada palabra subsiguiente es mayúscula.

Uso Principal

Nombres de variables y funciones. Es la convención dominante en JavaScript, Java, y muchos otros lenguajes.

Ejemplos de Código

let nombreDeUsuario = "Ana";
function calcularTotalVenta() { /* ... */ }

Pascal Case EjemploDeClase

Formato

La primera letra de todas las palabras está en mayúscula, incluida la primera.

Uso Principal

Nombres de clases, interfaces, componentes y constructores. Muy común en Java, C# y para clases en Python.

Ejemplos de Código

public class UsuarioService { /* ... */ }
// En React
function ComponenteDeNavegacion() { /* ... */ }

Snake Case ejemplo_de_variable

Formato

Todas las letras son minúsculas y las palabras están separadas por un guion bajo (_).

Uso Principal

Nombres de variables y funciones. Es la convención principal en Python y Ruby. También se usa a menudo en bases de datos SQL.

Ejemplos de Código

edad_del_cliente = 30
def guardar_registro_log():
    pass

Screaming Snake Case EJEMPLO_DE_CONSTANTE

Formato

Todas las letras están en mayúscula y las palabras están separadas por un guion bajo (_).

Uso Principal

Constantes globales o valores inmutables que no deben cambiar durante la ejecución del programa.

Ejemplos de Código

const MAX_CONEXIONES = 10;
// En Python
TASA_INTERES_BASE = 0.05

Kebab Case ejemplo-de-selector

Formato

Todas las letras son minúsculas y las palabras están separadas por un guion medio (-).

Uso Principal

Nombres de archivos, URLs (slugs), selectores de clase en CSS y atributos personalizados en HTML.

Ejemplos de Código

/* archivo: estilos-globales.css */
.boton-principal {
  background-color: blue;
}

Flat Case ejemplodevariable

Formato

Todas las letras en minúsculas sin ningún separador entre palabras.

Uso Principal

Poco común para variables largas por su baja legibilidad. A veces se usa para nombres de paquetes o variables muy cortas y simples.

Ejemplos de Código

# Un paquete podría llamarse 'usuariomodelo'
nombreusuario = "juanp"

Notación Húngara strEjemplo

Formato

Se añade un prefijo en minúscula a la variable para indicar su tipo de dato (ej. 'str' para string, 'i' para integer).

Uso Principal

Históricamente usada en C o C++. En lenguajes modernos con tipado fuerte, está en gran medida obsoleta y se considera mala práctica.

Ejemplos de Código

char* strNombre = "Carlos";
int iContador = 0;
