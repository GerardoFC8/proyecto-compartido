Explorador de Comandos Artisan

Una guía interactiva de los comandos de Artisan más comunes en Laravel. Navega por las categorías para encontrar rápidamente el comando que necesitas.

Estos comandos te ayudan a generar rápidamente la estructura de tu aplicación, como controladores, modelos y migraciones.

make:controller

Crea un nuevo controlador.

Opciones comunes:

  • -m: Crea un modelo.
  • -r: Crea un controlador de recursos.
  • --api: Crea un controlador de API.

make:model

Genera una nueva clase de modelo Eloquent.

Opciones comunes:

  • -m: Crea una migración.
  • -c: Crea un controlador.
  • -s: Crea un seeder.
  • -a: Crea todo lo anterior.

make:migration

Crea una nueva migración de base de datos.

Opciones comunes:

  • --create=users
  • --table=posts

make:seeder

Crea una nueva clase seeder.

make:factory

Genera una nueva clase factory.

make:request

Crea una nueva clase de formulario de solicitud.

make:middleware

Crea una nueva clase de middleware.

make:resource

Crea una nueva clase de recurso de API.

Opciones comunes:

  • --collection: Crea un recurso de colección.

make:component

Crea un nuevo componente Blade.

Administra el esquema de tu base de datos y puebla tus tablas con datos de prueba utilizando estos comandos.

migrate

Ejecuta las migraciones pendientes.

Opciones comunes:

  • --seed: Ejecuta los seeders.
  • --force: Permite ejecutar en producción.

migrate:rollback

Revierte la última tanda de migraciones.

Opciones comunes:

  • --step=5: Revierte 5 migraciones.

migrate:fresh

Elimina todas las tablas y ejecuta las migraciones desde cero.

Opciones comunes:

  • --seed: Ejecuta los seeders.

db:seed

Ejecuta los seeders para popular la base de datos.

Opciones comunes:

  • --class=UserSeeder

migrate:status

Muestra el estado de cada migración.

Gestiona el servidor de desarrollo local y la caché de tu aplicación para optimizar el rendimiento.

serve

Inicia el servidor de desarrollo de Laravel.

Opciones comunes:

  • --host=0.0.0.0
  • --port=8080

cache:clear

Borra la caché de la aplicación.

config:cache

Crea un archivo de caché para la configuración.

route:cache

Crea un archivo de caché para las rutas.

view:cache

Crea un archivo de caché para las vistas.

Comandos útiles para pruebas, modo de mantenimiento y la consola interactiva de Laravel.

test

Ejecuta las pruebas de la aplicación.

Opciones comunes:

  • --filter=MyTest

down

Pone la aplicación en modo de mantenimiento.

up

Saca la aplicación del modo de mantenimiento.

tinker

Inicia la consola interactiva de Laravel (REPL).

list

Muestra todos los comandos de Artisan disponibles.

Aquí tienes ejemplos prácticos para combinar comandos y opciones, acelerando tu flujo de trabajo de desarrollo.

Generar un controlador de recursos

Crea un controlador con métodos para operaciones CRUD básicas (index, create, store, show, edit, update, destroy).

php artisan make:controller PostController -r

Generar un "todo" para un modelo

Crea un modelo, su migración, un controlador de recursos, factory y seeder con un solo comando.

php artisan make:model Product -a

Refrescar la base de datos y sembrarla

Ideal para desarrollo, este comando borra todas las tablas, ejecuta las migraciones y luego puebla la base de datos con los seeders.

php artisan migrate:fresh --seed

Iniciar el servidor en un puerto específico

Útil cuando el puerto por defecto (8000) está ocupado.

php artisan serve --port=8080
