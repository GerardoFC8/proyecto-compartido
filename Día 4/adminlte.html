Potencial Completo de Laravel con AdminLTE

Bienvenido al explorador interactivo del paquete jeroennoten/laravel-adminlte. Esta aplicación transforma la documentación estática en una experiencia práctica. Navega por las secciones en el menú de la izquierda para descubrir cómo este paquete te proporciona un framework completo para construir paneles de administración robustos y funcionales, desde la configuración básica hasta la creación de dashboards complejos.

Empieza a explorar

Haz clic en cualquier sección del menú para ver demostraciones en vivo de las características y componentes.

Configuración y Personalización

La personalización se realiza principalmente en config/adminlte.php. Puedes modificar desde el título y logo hasta el layout completo del panel.

Título y Logo

Define la identidad visual de tu panel, incluyendo el texto y las imágenes que representan tu marca.

Skins (Pieles)

Elige entre varios esquemas de color predefinidos (skin-blue, skin-purple, etc.) para adaptar la apariencia.

Layout (Diseño)

Ajusta el comportamiento del layout: sidebar fijo, barra de navegación fija, diseño en caja ("boxed") y más.

URL y Rutas

Personaliza las URLs para las acciones de autenticación como inicio de sesión, registro y cierre de sesión.

Constructor de Menú Dinámico

Una de las características más potentes es el constructor de menú en config/adminlte.php, que permite añadir items, submenús, iconos y lógica de autorización con los Gates de Laravel.

Ejemplo de Configuración


'menu' => [
    'MAIN NAVIGATION',
    [
        'text'    => 'Blog',
        'url'     => 'admin/blog',
        'can'     => 'manage-blog',
        'icon'    => 'far fa-fw fa-file-alt',
    ],
    [
        'text'    => 'Pages',
        'icon'    => 'far fa-fw fa-file',
        'submenu' => [
            [
                'text' => 'Page 1',
                'url'  => '#',
            ],
        ],
    ],
],

Integración de Plugins Populares

Habilita plugins fácilmente de forma global o por página con @section('plugins.NombrePlugin', true) para añadir funcionalidades avanzadas.

Chart.js

Crea gráficos y visualizaciones de datos. A continuación un ejemplo en vivo.

SweetAlert2

Reemplaza los `alert()` con modales elegantes y personalizables.

DataTables

Añade controles avanzados a tus tablas HTML, como búsqueda, paginación y ordenamiento.

Select2

Mejora los campos `select` de tus formularios con capacidades de búsqueda y etiquetado.

Integración con el Ecosistema Laravel

AdminLTE funciona perfectamente con herramientas modernas del ecosistema de Laravel, permitiéndote crear aplicaciones complejas y dinámicas.

Livewire

Construye componentes dinámicos y reactivos que se integran a la perfección en tus vistas de AdminLTE, creando paneles en tiempo real sin recargar la página.

Inertia.js

Utiliza AdminLTE como el layout principal para una Single-Page Application (SPA) moderna construida con Inertia, Vue o React.
